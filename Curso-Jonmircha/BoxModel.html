The box model is a way of representing the elements of a web page in terms of boxes. It is used to define the size, position, and appearance of elements on a web page. The box model consists of four parts: the content, the padding, the border, and the margin. The content is the actual text or image that is displayed inside the element. The padding is the space between the content and the border. The border is the line that surrounds the element. The margin is the space between the element and other elements on the page. The box model is used to control the layout of elements on a web page. By changing the size, position, and appearance of the boxes, you can create different layouts for your web pages.

Padding

The padding is the property that allows adding space internally to the box that contains the element.

When we define the padding in a general way, a shorthand is being applied, which is the general value for the 4 sides of the box. 

                
                      padding: 10px;

When we define 2 values, the first value will always be for the vertical dimensions (top-bottom) and the second for the horizontal ones (left-right).

                
                    padding: 10px 20px;

When we define 4 values, they are taken for the 4 sides chronologically clockwise. top-right-bottom-left.



Margin

The margin is the external spaces around other elements.

Just like the padding and the border, the margin can also be defined in a general way and this makes a shorthand, that is, it takes a single value for the 4 sides of the box.

                
                    margin: 10px;

On the other hand we can also define the shorthand in such a way that we give 2 values and 4 values at the same time.

                
                    margin: 10px 20px;
                    margin: 10px 20px 30px 40px;

Block Box vs Online Box

They are elements that serve as containers for HTML elements.

In block

  • Occupy as wide as possible.

  • They have modifiable dimensions (height, width)

  • Don't allow other elements next to it (even if you specify a width, they still take up all available space)

In line

  • Occupy the necessary space to display its content.
  • They do not have modifiable dimensions (height, width).
  • Allow other items to your side
  • Padding and margin only push adjacent elements horizontally, never VERTICALLY.

Propertie visibility

It is the property that allows an element to be displayed or not, by default all the elements bring it since they are always displayed (visible), but so that they are not seen, the value of hidden is placed, it will not be seen but the space it occupies continues to be displayed .


Propertie overflow

It is used to control the behavior of content that exceeds the size of the container to which it belongs. That is, when the content within an HTML element is larger than the space available to it, the overflow property determines how that additional content will be handled.

The overflow property can take several values:

  1. visible (default): Content that overflows the container is displayed outside the container, that is, it overflows the container. It can overlap other elements if necessary.

  2. hidden: All the content that overflows the container is hidden, that is, it is cut off and not shown.

  3. scroll: Both horizontal and vertical scroll bars will appear, allowing the user to scroll the content to see the hidden area.

  4. auto: Similar to scroll, it shows scrollbars only when necessary. If there is no overflowing content, the bars will not be displayed.

  5. inherit: Inherits the value of the overflow property from the parent element.
