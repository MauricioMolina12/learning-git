Basic Selectors

Basic selectors in CSS are those that allow you to select an element or a group of elements on a web page. The most commonly used selectors are:

  • The element selector, which selects all elements of a specific type. For example, the selector p selects all paragraphs on the page.
  • The class selector, which selects all elements that have a specific class. For example, the selector .myclass selects all elements with the class myclass.
  • The ID selector, which selects the element with a specific ID. For example, the selector #myid selects the element with the ID myid.

These are the basic selectors, but there are many other more advanced selectors that allow for more precise element selection.



Advanced Selectors

Sons's selectors

    Direct sons selectors

    Child Selectors are those that select elements that are direct children of the current element. For example, if we have the following code:

                                    
                                    <div>
                                                <p>This is a paragraph </p>
                                                <p>This is another paragraph </p>
                                      </div>

    And we want to select the first paragraph, we can use the following selector:

                                    
                                    div > p:first-child

    This will select the first paragraph inside the `div` element.

    Descendant sons selector

    Child Descendant Selectors are those that select elements that are descendants of the current element. For example, if we have the following code:

                                    
                                    <div>
                                                <p>This is a paragraph </p>
                                                <p>This is another paragraph </p>
                                      </div>

    And we want to select all paragraphs, we can use the following selector:

                                    
                                    div > p

    This will select all paragraphs inside the `div` element.
