Basic Selectors

Basic selectors in CSS are those that allow you to select an element or a group of elements on a web page. The most commonly used selectors are:

  • The element selector, which selects all elements of a specific type. For example, the selector p selects all paragraphs on the page.
  • The class selector, which selects all elements that have a specific class. For example, the selector .myclass selects all elements with the class myclass.
  • The ID selector, which selects the element with a specific ID. For example, the selector #myid selects the element with the ID myid.

These are the basic selectors, but there are many other more advanced selectors that allow for more precise element selection.

Universal Selector

The universal selector is *. It matches every element in the document.

                    
                        *{

Pseudoclasses

Pseudo-classes are special keywords that you can use in CSS to style elements based on their state or position. For example, you can use the :hover pseudo-class to style an element when the user hovers over it, or the :active pseudo-class to style an element when the user clicks on it.

Cascada

Cascada is a CSS property that specifies the order in which CSS rules are applied to an element. It can be used to control the stacking order of elements, and to ensure that certain rules are applied before others. The cascading order is determined by the following factors:

  • The order in which the rules are written in the CSS file.
  • The specificity of the rules.
  • The importance of the rules.
The specificity of a rule is determined by the number of selectors it contains. A rule with a single selector has a lower specificity than a rule with multiple selectors. The importance of a rule is determined by its position in the CSS file. Rules that are written later in the file have a higher importance than rules that are written earlier. The cascading order can be changed by using the `!important` declaration. This declaration overrides the normal cascading order and ensures that the rule is applied before all other rules.

Especificidad

In CSS (Cascading Style Sheets), specificity is a concept that determines which rules will be applied to an element when there are multiple rules trying to style the same element. When conflicts occur between rules, the browser uses specificity to decide which one will take precedence.

Specificity is based on the combination of selectors used in a CSS rule. Each selector has a specificity value calculated based on its type and number of components. Generally, we can establish the specificity weight of selectors as follows (from lower to higher specificity):

  • Etiquetas y Pseudoelementos------------------------- 0,0,0,1
  • Clases, Atributos y Pseudoclases-------------------- 0,0,1,0
  • Identificadores--------------------------------------------- 0,1,0,0
  • Estilos en línea-------------------------------------------- 1,0,0,0
  • !important-------------------------------------------------Rompe la especificidad

Inheritance

Inheritance in CSS is the ability to pass properties from one element to another. This can be used to create a consistent look and feel across your website, and to save time by reusing styles.

To inherit a property, simply add the `inherit` keyword to the property value. For example, if you want to inherit the `font-size` property from the `body` element to the `h1` element, you would add the following code to your CSS:

                            
                                h1 {
                                    font-size: inherit;
                                  }

This would cause the `h1` element to use the same font size as the `body` element. Inheritance can also be used to inherit the values of multiple properties at once. To do this, simply list the properties separated by commas, and add the `inherit` keyword to the end of the list. For example, the following code would cause the `h1` element to inherit the `font-size`, `font-family`, and `color` properties from the `body` element:

                            
                                h1 {
                                    font-size: inherit;
                                    font-family: inherit;
                                    color: inherit;
                                  }
